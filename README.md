# Unemployment Analysis with Python

## Overview

This project analyzes unemployment trends in India using data analysis and visualization techniques to understand employment patterns across regions and over time.

## Objective

To analyze unemployment data, identify trends and regional differences, and present insights through visualizations.

## Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset Features

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Labour Participation Rate (%)

## Steps Performed

1. Loaded the unemployment dataset.
2. Cleaned and prepared the data.
3. Converted date values into proper format.
4. Performed exploratory data analysis.
5. Analyzed unemployment trends across regions.
6. Created visualizations for better understanding.
7. Compared unemployment rates among states.
8. Interpreted the findings.

## Visualizations

* Line Chart for Unemployment Trends
* Bar Chart for State-wise Comparison

## Outcome

The project successfully identified unemployment patterns and regional differences, providing meaningful insights through data visualization and analysis.
